# AI in Health Informatics – CKD Prediction

This project investigates Chronic Kidney Disease (CKD) prediction using Machine Learning, Deep Learning, and Explainable AI.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- Gradient Boosting
- XGBoost
- LSTM
- GRU

## Explainable AI
- SHAP
- LIME
- Feature Importance Analysis
